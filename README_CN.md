# 🛠️ GC-Fix

## Language Options

- [English](README.md)
- 中文

## 📋 概述

GC-Fix是一个轻量级Windows工具，专为解决"GC-PowerStation"软件的启动问题而设计。通过重置应用程序的注册表设置，它可以恢复默认UI布局并解决常见的启动故障。

## ✨ 核心功能

- **极小体积**：仅6KB大小，性能高度优化
- **操作简单**：一键执行，带有清晰的确认消息
- **安全注册表处理**：安全地递归删除注册表项
- **自定义图标**：独特图标，易于识别
- **便携性**：无需安装，可从任何位置运行
- **32位兼容**：适用于所有Windows版本

## 🚀 快速开始

1. 将 `GC-Fix.exe` 下载到您的计算机
2. 双击启动应用程序（无需安装）
3. 点击"确定"确认注册表重置操作
4. 等待成功确认消息
5. 重新启动GC-PowerStation查看恢复的默认UI布局

## 🛠️ 技术规格

| 参数 | 详情 |
|------|------|
| **架构** | 32位Windows可执行文件 |
| **开发语言** | 汇编语言 (MASM32) |
| **文件大小** | 6KB |
| **依赖项** | 无（静态链接） |
| **图标来源** | https://www.iconfont.cn/ |
| **构建工具** | MASM32汇编器、资源编译器 |

## 📦 构建说明

1. 确保MASM32已安装在 `D:\masm32`
2. 在项目目录中打开命令提示符
3. 执行 `build.bat` 编译并链接可执行文件
4. 生成的 `GC-Fix.exe` 将在同一目录中创建
